# Food Tracking Guide

A single-file, self-contained web guide on food tracking — honestly.

**Title:** Food Tracking Guide — Myths, Mistakes & Practical Caveats

Why precision is a myth, the few execution mistakes that actually matter (raw vs. cooked, eyeballing, the "little stuff," weekend blowouts), and how to track in a way you'll actually keep. Evidence-based. No filler.

## Files

- `index.html` — the entire guide. All HTML, CSS, and JS inline. No dependencies, no build step.

## Use

Open `index.html` in any browser, or deploy the repo to any static host (Vercel, GitHub Pages, Netlify) — it serves at the root with zero configuration.
